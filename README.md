# 🎬 Movie Recommendation System

A **Content-Based Movie Recommendation System** built using **Python**, **Machine Learning**, and **Streamlit**. This application recommends the **Top 10 similar movies** based on the selected movie and displays their posters using the **TMDB API**.

---

## 🚀 Features

- 🎥 Recommend Top 10 Similar Movies
- 🖼️ Display Movie Posters using TMDB API
- 🤖 Content-Based Recommendation using Cosine Similarity
- ⚡ Interactive Streamlit Web Application
- 📊 Fast and Accurate Recommendations

---

## 📖 How It Works

1. Load the movie dataset.
2. Preprocess the movie information.
3. Create feature vectors from movie metadata.
4. Calculate cosine similarity between movies.
5. Recommend the Top 10 most similar movies.
6. Fetch movie posters using the TMDB API.
7. Display the recommendations through a Streamlit web application.

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/Moulika-tata/Movie_recommendation_system.git
```

Move into the project folder:

```bash
cd Movie_recommendation_system
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
